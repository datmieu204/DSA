    f5 = 4n => O(n)
    f6 = n^3 => O(n^3)
    f7 = 2^logn = n => O(n)
    f8 = 2^n => O(2^n)
    f9 = nlogn => O(nlogn)

    Sorted: f4 < f7 < f5 < f2 < f9 < f1 < f3  < f6 < f8
